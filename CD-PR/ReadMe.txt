-----------------------EN------------------------

1. First, open the "main.ipynb" file.
2. Here, "!pip install easyocr" and other libraries should be installed in the top block. 
    (For installation from the terminal, the beginning of pip should be "pip", not "!pip".) 
3. Open the "PreProcessing.py" file in the same program. Make sure that the selected python kernels are the same.
4. First, run the PreProcessing.py file, but there will be no output. You just need to run it.
5. Go back to the "main.ipynb" file and run the import required areas in the second block.
6. When the import process is completed without any errors, the main code block is executed.
7. "input Video" should be located in the same location of your Python file. This is an example input trial video.
8. Video processing and output takes approximately 7-8 minutes.
9. When the code runs successfully, the output video will be written to the "OutputVideo" folder.
10. All detected tools can be seen in the "OutputImage" folder. For each ID, recording was made starting from the 15th frame.



-----------------------TR------------------------

1. Öncelikle "main.ipynb" dosyasını açın.
2. Burada en üst bulunan blokta "!pip install easyocr" ve diğer kütüphaneler kurulmalıdır. 
   (Terminalden kurulum için pip başındaki "!pip" değil "pip" olmalıdır.) 
3. "PreProcessing.py" dosyasını yine aynı programda açın. Seçilen python kernellerin aynı olmasına dikkat ediniz.
4. Öncelikle PreProcessing.py dosyasını çalıştırın fakat çıktı gelmeyecektir. Hata verebilir fakat sadece çalıştırmanız yeterlidir. Hataları görmezden gelebilirsiniz.
5. "main.ipynb" dosyasına geri gidin ve ikici bloktaki import gereken yerleri çalıştırın.
6. Hatasız bir şekilde import işlemlerini tamamladığı zaman ana kod bloğu çalıştırılır.
7. Python dosyanızın aynı konumunda "inputVideo" bulunmalıdır. Bu örnek giriş videosudur.
8. Video işleme ve çıktı verme süresi yaklaşık olarak 7-8 dk sürmektedir.
9. Kodun çalışması başarıyla bitince çıkış videosu "OutputVideo" klasörü içine yazılacaktır.
10. Bütün tespit edilmiş araçlar "OutputImage" klasöründe görülebilir. Her ID için 15. Frameden başlayarak kayıt yapılmıştır.
